# iris-dashboard
# 🌸 Iris Flower Data Analysis Dashboard

## 📌 Project Overview
This project is an interactive data visualization dashboard built using Streamlit.  
It analyzes the Iris dataset and provides insights through multiple charts and filters.

---

## 📊 Dataset Information
The dataset used is the classic Iris dataset containing:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width
- Species (Setosa, Versicolor, Virginica)

---

## 🎯 Objectives
- Perform Exploratory Data Analysis (EDA)
- Visualize data using multiple chart types
- Build interactive filters
- Present insights in a professional dashboard

---

## 📈 Features

### Charts Included:
- Pie Chart (Species distribution)
- Histogram (Feature distribution)
- Scatter Plot (Feature relationships)
- Heatmap (Correlation matrix)
- Box Plot (Outliers & spread)
- Bar Chart (Category comparison)
- Line Chart (Trend visualization)
- Area Chart (Cumulative view)
- Count Plot (Frequency analysis)
- Violin Plot (Distribution shape)

### Filters:
- Species selection
- Numerical range slider (Sepal Length)
- Dynamic filtering across all charts
